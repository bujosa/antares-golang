> Antares Golang is a project about code organization, go and Postgres.

## Overview

**Primary language:** Go

## Tech Stack

- **Frameworks / Libraries:** Postgresql
- **Data:** Postgresql

## Getting Started

```bash
git clone https://github.com/bujosa/antares-golang.git
cd antares-golang
go mod download
go run .
```
